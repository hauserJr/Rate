# Rate_Demo
查詢各匯率


使用兩種套件

1.[HtmlAgilityPack](https://html-agility-pack.net/)

  用於解析網頁內容(XPATH)

2.[ILMerge](https://github.com/dotnet/ILMerge)

  整合多個組件，可參考[保哥：介紹好用工具：ILMerge (將多的 .NET 組件合併成單一組件)](https://blog.miniasp.com/post/2009/08/07/Useful-tool-ILMerge)


# 使用說明

1. 每次查詢區間為15分鐘。

2. 目前僅開放台新銀行(TaiShin)
